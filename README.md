## ‚ú® Features
- Search *(move the cursor on the app and type)*
- Scrolling
- Open / Close the frame
- Background styles: Acrylic, Black or white with transparency
- Support for showing/hiding hidden files
- Sticky frames
- Option to lock Frames
- And many more
